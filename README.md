## 導入方法(ローカル)

```bash
npm install
```

環境変数 (.env ファイル)

```bash
DB_USER=
DB_PASSWORD=
DB_NAME='mvp_db'
```

## コマンド一覧

| コマンド | 実行される処理           |
| -------- | ------------------------ |
| start    | サーバー起動(nodemon)    |
| dev      | フロントエンド起動(vite) |
| migrate  | migrate の実行           |
| seed     | seed の実行              |
| render   | render デプロイ用        |
